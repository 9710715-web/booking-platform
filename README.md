# Booking Platform — Статический сайт

Этот репозиторий содержит простую HTML‑страницу для размещения на Vercel.

## Настройки проекта на Vercel
1. Открой вкладку **Settings → Build & Output Settings**.
2. Установи параметры:
   - Framework Preset: **Other**
   - Build Command: **(пусто)**
   - Install Command: **(пусто)**
   - Output Directory: **.**
3. Нажми **Save**.
4. Перейди во вкладку **Deployments** и нажми **Redeploy**.

После этого сайт откроется по адресу `https://booking-platform.vercel.app/`.
